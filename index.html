Intro to SASS

What is SASS?

Sass is CSS with Superpowers

It is a CSS preprocessor.

  • Been around for 8 years
  • Actively supported
  • Completely compatible with all versions of CSS
  • Made by Hampton Catlin, Natalie Weizenbaum, and Chris Eppstein

Using SCSS Partials we can separate large complex stylesheets into sane, modular pieces.

to this

  • _typography.scss (50 lines)

Mixins(functions)

Functions & Parameters


@mixin border-radius($radius) {
  -webkit-border-radius: $radius;
     -moz-border-radius: $radius;
      -ms-border-radius: $radius;
          border-radius: $radius;
}

.box { @include border-radius(10px); }
                            

.box {
    -webkit-border-radius: 10px;
    -moz-border-radius: 10px;
    -ms-border-radius: 10px;
    border-radius: 10px;
}

Extend/Inheritance

Keep your SASS DRY


.message {
    border: 1px solid #ccc;
    padding: 10px;
    color: #333;
}

.success {
    @extend .message;
    border-color: green;
}
                            

.message, .success {
    border: 1px solid #cccccc;
    padding: 10px;
    color: #333;
}

.success {
  border-color: green;
}

Practical Example 5

Oh no!
media queries!

So make a mixin.


@mixin media($type) {
    @if $type == large-desktop {
        @media only screen and (min-width: 1200px) {
            @content;
        }
    }
    @else if $type == desktop {
        @media only screen and (max-width: 1199px) {
            @content;
        }
    }
}
                            

.desktop .chatBlock {
    @include media(large-desktop) {
        margin-top: 2em;
    }
}
