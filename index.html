Mixins(functions)

Functions & Parameters


@mixin border-radius($radius) {
  -webkit-border-radius: $radius;
     -moz-border-radius: $radius;
      -ms-border-radius: $radius;
          border-radius: $radius;
}

.box { @include border-radius(10px); }
                            

.box {
    -webkit-border-radius: 10px;
    -moz-border-radius: 10px;
    -ms-border-radius: 10px;
    border-radius: 10px;
}

Extend/Inheritance

Keep your SASS DRY


.message {
    border: 1px solid #ccc;
    padding: 10px;
    color: #333;
}

.success {
    @extend .message;
    border-color: green;
}
                            

.message, .success {
    border: 1px solid #cccccc;
    padding: 10px;
    color: #333;
}

.success {
  border-color: green;
}

Practical Example 5

Oh no!
media queries!

So make a mixin.


@mixin media($type) {
    @if $type == large-desktop {
        @media only screen and (min-width: 1200px) {
            @content;
        }
    }
    @else if $type == desktop {
        @media only screen and (max-width: 1199px) {
            @content;
        }
    }
}
                            

.desktop .chatBlock {
    @include media(large-desktop) {
        margin-top: 2em;
    }
}

Compass is an open-source CSS Authoring Framework.
http://compass-style.org/

Compass is a powerful set of mixins, functions, and basically SASS extensions.

  1. Experience cleaner markup without presentational classes.
  2. It’s chock full of the web’s best reusable patterns.
  3. It makes creating sprites a breeze.
  4. Compass mixins make CSS3 easy.
  5. Create beautiful typographic rhythms.
  6. Download and create extensions with ease.

Installation 1

The daunting setup!

Install the following:

  1. Install Ruby - Ruby 2.0.0-p643 (x64)
  2. Install Python - Python-2.7.9

In the installer(s), be sure to add to environment variables.

Open command prompt, check versions.


ruby -v
gem -v
python

Install the required gems.


gem install sass
gem install compass

Do it.

Installation 2

Lets setup a simple SASS project, using the command line.

1. Go to your desired project location and make a directory.


cd %USERPROFILE%/desktop
mkdir sass-test

2. Cd into it, and create your first sass project.


cd sass-test
compass create

3. Watch for changes.


compass watch

Do it.
